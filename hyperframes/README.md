# DeepBook Risk Console HyperFrames Pitch Demo

This folder contains a rebuilt product-led HyperFrames composition for the judge-facing demo video.

The composition follows the supplied reference more closely: dark product page, kinetic product cards, large bilingual subtitles, and full-screen local app proof moments.

## Commands

```bash
npx hyperframes lint .
npx hyperframes inspect . --samples 14
npx hyperframes render . --output ../media/deepbook-risk-console.hyperframes.mp4 --quality draft --workers 1
```
